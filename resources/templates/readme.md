# TEMPLATES

## WHAT THESE ARE
Blank, reusable starting points for documenting and shipping skills! :)

## HOW TO USE
1. COPY THE TEMPLATE YOU NEED INTO YOUR WORKING DIRECTORY.
2. RENAME IT (REMOVE `_TEMPLATE`).
3. FILL EVERY PLACEHOLDER SECTION.
4. KEEP HEADINGS IN ALL CAPS FOR CONSISTENCY.
5. KEEP THE ATTRIBUTION FOOTER INTACT.
